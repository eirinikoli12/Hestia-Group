What is the use of Properties button in homepage?
The first button is used to have access to the properties that Hestia-Group already owns. Some members of the company are obliged to set a budget and update property status. Via this button you can search and update for property records, approve and set a budget.
What is the use of Set a Task button in homepage?
This button was created mostly for the Marketing Department. Specifically, Marketing and Sales Manager has the responsibility to set tasks to her Assistants. However, this button can also be used by the CEO and use this way to set tasks to employees instead of WeChat.
What is the use of Read a Task button in homepage?
With this button members are able to see clearly and in an organised way all of the tasks that they need to accomplish.
What is the use of Notifications/News button in homepage?
With this button members stay up-to-date with all of the important information that they need to know. Also, they are able to stay connected with the Chinese Company, Taibo Investment.
What is the use of Information button in homepage?
In this button is shared all of the information on how to use this Information System efficiently.
